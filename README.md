# سیستمی بەڕێوەبردنی لیژنەکانی قائیمقامیەت
## سیستەمی لۆکاڵی بۆ ٥٠+ ئایپاد

### شێوازی دانان:

#### ١. ئامادەکاری سیستەمی لۆکاڵی WiFi:
- ڕاوەستەرێکی WiFi دانێ (Router)
- کۆمپیوتەری ئەدمین پەیوەست بکە بە WiFi
- هەموو ئایپادەکان پەیوەستی بە هەمان WiFi بکە

#### ٢. دانانی سیستەم لەسەر کۆمپیوتەری ئەدمین:
- فایلی index.html بکەرەوە لەسەر کۆمپیوتەری ئەدمین
- سێرڤەری لۆکاڵی دابنێ بەم فەرمانە:
```
python -m http.server 8080
```
یان
```
npx serve -s . -l 8080
```

#### ٣. چوونەژوورەوە لەسەر ئایپادەکان:
- لەسەر هەر ئایپادێک ئەم ناونیشانە بنووسە:
```
http://[IPی_کۆمپیوتەر_ئەدمین]:8080
```
- بۆ دۆزینەوەی IPی کۆمپیوتەری ئەدمین:
  - لە Windows: `ipconfig` لە Command Prompt
  - لە Mac/Linux: `ifconfig` لە Terminal

### چالاکی سیستەم:
- ✅ ٥٠+ ئایپاد دەتوانن هەمان کات کاری پێبکەن
- ✅ هەر یوزەرێک داتا تۆمار دەکات، ڕاستەوخۆ لای ئەدمین دەردەکەوێت
- ✅ پێویست بە ئینتەرنێت نییە - تەنها WiFi لۆکاڵی
- ✅ داتا لەسەر کۆمپیوتەری ئەدمین پاشەکەوت دەبێت
- ✅ ئەگەر WiFi نەمەینەوە، داتا لەسەر هەر ئایپادێک پاشەکەوت دەبێت

### شێوازی بەکارهێنان:
١. ئەدمین سیستەمەکە دەکاتەوە
٢. یوزەرەکان دروست دەکات (بۆ هەر ئایپادێک یەک یوزەر)
٣. هەر ئایپادەوە بە یوزەرەکەی چوونەژوورەوە دەکات
٤. دەست بە تۆمارکردنی داتا دەکەن
٥. ئەدمین ڕاستەوخۆ هەموو داتاکە دەبینێت

### نرخی سیستەم:
- 🆓 تەواو فرییە
- 💻 پێویستە تەنها یەک کۆمپیوتەر بێت
- 📱 پشتیوانی لە هەموو ژمارە ئایپادێک دەکات
- 🌐 پێویست بە ئینتەرنێت نییە

### کێشەکان و چارەسەرەکان:
- **ئەگەر WiFi نەمێنێتەوە:** سیستەم بەردەوام دەبێت لەسەر هەر ئایپادێک
- **ئەگەر کۆمپیوتەر داخرا:** داتا لەسەر ئایپادەکان ماوە
- **یەکخستنی داتا:** کاتێک WiFi هەیە، هەموو داتا یەکدەبێتەوە خۆکارانە
